# **神经生物学与脑科学**

## **课程内容**
本课程中将学习神经元的结构和工作原理，探索神经回路如何处理信息、存储记忆和控制行为。重点关注神经网络、学习机制以及感知编码等主题，为理解人工智能、机器学习和类脑计算提供扎实的生物学基础，帮助能从生物学视角汲取灵感，设计更先进的计算模型。

## **学习建议**
建议听老师上课的重点，当作纯生物课来学，重点部分可以参考神经宝典，答题基本上就是按照上课讲过的重点词汇来答，考前突击最好也能把知识点的概念背下来。
