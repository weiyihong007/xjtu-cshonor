# **管理学基础(弃用)**

## **课程内容**
本课程将系统性地介绍管理学的四大职能：计划、组织、领导和控制。通过学习，掌握决策、团队协作、激励员工等核心技能，并了解企业社会责任与道德规范，打下扎实的商业思维和领导力基础。

## **学习建议**
建议抓住课内的内容，考题基本上就是上课提到的东西，术语会有倾斜，没有听课的分数最后出来都差不太多。
