# **算法设计与分析**

## **课程内容**
课程将系统地介绍设计高效算法的核心思想，包括分治法、动态规划、贪心算法、回溯法等。学会用数学方法来严谨地分析和证明算法的时间复杂度与空间复杂度，并掌握NP完全性理论。打好这门课的基础是成为优秀工程师或研究人员的关键能力。

## **学习建议**
课本上的例题对于部分同学来说可能会比较难，但是最终考试题可能还是以课后题为准(但老师也有可能整花活)。建议课外自主多拓展一下这方面的知识，亦可以向由相关方面知识的同学请教。
